Tapestry is a powerful, all-Java framework for creating leading edge web applications in Java. Tapestry is being developed by Howard Ship of Primix Solutions.

Tapestry is positioned primarily as an alternative to JavaServer Pages. It replaces the scripting and code generation of JSPs with a full-fledged component object model, allowing for a very high degree of code reuse, in the form of reusable components.
