InsertBlock
net.sf.tapestry.components.InsertBlock
Non Visual Component
 
Description
Renders the text and components of the specified Block component.

Please see the InsertBlock API for a complete discription on how the InsertBlock renders.

This component may NOT wrap around other components or static HTML.

See Also
Block, Insert, InsertText, InsertWrapped
Parameters
Name Type Direction Required Default Description
block Block in no   The Block whose contents are to be rendered.
Informal parameters: allowed
Reserved parameters: none
