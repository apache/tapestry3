GenericLink
net.sf.tapestry.link.GenericLink
Generic Link
Description
Creates a HTML <a> hyperlink for an arbitrary URL.

Although a GenericLink can be accomplished using an Any component, a GenericLink is more flexible: it can be disabled, like a Direct or Action and can also wrap around Rollover components. GenericLinks are often used with javascript: URLs to perform client-side scripting.

This component may wrap around other components and static HTML.

See Also
Action, Direct, Page, Rollover, Service
Parameters
Name Type Direction Required Default Description
href String in yes   The URL to trigger when the link is clicked. This is often of the form:
<a href="javascript:functionName('param')">
disabled boolean in no false Controls whether the link is produced. If disabled, the portion of the template the link surrounds is still rendered, but not the link itself.
Informal parameters: allowed
Reserved parameters: none
