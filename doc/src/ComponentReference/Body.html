Body
net.sf.tapestry.components.Body
Visual / Non Visual Component
Description
Replaces the <body> tag. The Body component provides support dynamic scripting and image preloaded, used by other components, such as the Rollover component.
See Also
Rollover, Script
Parameters    (informal parameters allowed)
Name Type Direction Required Default Description
name String in no body The element to use when rendering a tag. When using a Body component in a frameset, this may be changed to "frameset".
Examples

The Body component is used here to dynamically set the page's background color using the informal parameter 'bgcolor'.

Your account of $645.35 is 2 Months overdue. 

<body jwcid="body">
  Your account <span jwcid="insertAccountStatus"/>.
</body>


<component id="body" type="Body">
   <binding name="bgcolor" property-path="statusColor"/>
</component>

<component id="insertAccountStatus" type="Insert">
   <binding name="value" property-path="accountStatus"/>
</component>


public class AccountPage extends BasePage {
    private String accountStatus;
    private String statusColor;

    public String getAccountStatus() { return accountStatus; }

    public String getStatusColor() { return statusColor; }
    
    public void detach() {
        accountStatus = null;
        statusColor = null;
        super.detach();
    }
}
