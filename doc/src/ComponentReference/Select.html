Select
net.sf.tapestry.form.Select
Description
Implements a list box selection. The items shown within the selection can be sourced via a PropertySelection or from related Option components.

In most normal circumstances, the options to be selected are provided via PropertySelection.

This component may wrap around other components and static HTML.

See Also
Form, Option, RadioGroup PropertySelection
Parameters
Name Type Direction Required Default Description
multiple boolean in no false If true, the component allows multiple selection.
disabled boolean in no false Controls whether the select is active or not.

Corresponds to the disabled HTML attribute.
Informal parameters: allowed
Reserved parameters: none
