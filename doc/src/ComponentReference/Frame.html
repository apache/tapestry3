Frame
net.sf.tapestry.html.Frame
Visual / Non Visual Component
Description
Provides the HTML <frame> element within a <frameset> inserting the specified Tapestry page.

This component may NOT wrap around other components or static HTML.

See Also
InsertBlock, InsertWrapped
Parameters
Name Type Direction Required Default Description
page String in yes   The page to display in the frame.
Informal parameters: allowed
Reserved parameters: "src"
