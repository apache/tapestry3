ShowInspector
net.sf.tapestry.inspector.ShowInspector
 
Description
Adds a link to a page that is used to raise the Tapestry Inspector in a new window.

The Tapestry Inspector enables you at runtime to examine composition of a Tapestry application, inspect persistent page properties, clear HttpSessions and Tapestry caches, and modify Log4J logging configuration.

Typically, a ShowInspector is wrapped by a Conditional component, configured so that it only is displayed when some form of development mode flag is enabled.

Pages incorporating a ShowInspector component must also include a Body component.
