External Distributions

This directory contains external distributions that can no longer be included with Tapestry due to licensing conflicts. It starts empty -you must acquire the necessary distributions from the provided links and copy them into this directory before attempting to run the examples.

Also, don't forget to update config/build.properties to point at your Jetty or Tomcat installation (to run the Workbench), or at your JBoss installation (to run the Virtual Library).

File Source Description
mckoidb-0.94h.jar http://www.mckoi.com/database/ SQL Database used with the Virtual Library
mckoidb-0.94h-jmx.jar http://sourceforge.net/project/showfiles.php?group_id=4754 JMX MBean wrappers for McKoi, to run inside JBoss
