wml:Card
org.apache.tapestry.wml.Card
Non Visual Component
 
Description
A WML deck contains one or more cards. The wml:Card component defines a card. A card can contain text, markup, links, input-fields, tasks, images and more.

See Also
Parameters
Name Type Direction Required Default Description
title String in yes   The title attribute specifies advisory information about the element. The title may be rendered in a variety of ways by the user agent (eg, suggested bookmark name, pop-up tooltip, etc.).

Body: removed
Informal parameters: allowed
Reserved parameters: none
