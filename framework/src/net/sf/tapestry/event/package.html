Defines events and listener interfaces for Tapestry.

{@link net.sf.tapestry.event.ChangeObserver} and {@link net.sf.tapestry.event.ObservedChangeEvent} are used to communicate changes in persistent properties from pages and components to page recorders.

The remaining interfaces {@link net.sf.tapestry.event.PageCleanupListener}, {@link net.sf.tapestry.event.PageDetachListener} and {@link net.sf.tapestry.event.PageRenderListener} allow objects to know about key lifecycle events regarding a page. @author Howard Lewis Ship hlship@apache.org
