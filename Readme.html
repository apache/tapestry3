Tapestry Release 0.2.11

Tapestry is a component based framework for creating sophisticated web applications. It's used with existing application servers, acting as a kind of "super servlet". This distribution contains the Tapestry framework, related frameworks and libraries, documentation. The Tapestry Home Page contains much more information about what Tapestry is, how to use it and why you can't live without it!

Tapestry is distributed in three different sizes:

  • small -- just Tapestry.jar, gnu-regexp.jar, the Tapestry Developer's Guide and the javadoc for Tapestry
  • medium -- small, plus Tapestry source code, the JBE (Java Build Environment) and JBE documentation
  • full -- medium, plus third party JARs used by Tapestry, more Tapestry documentation, Tapestry tutorials and the Primix Virtual Library examples

The JARs were compiled on Windows 2000 using Sun's JDK 1.3 compiler, however it is easy to rebuild the JARs on different platforms and different compilers (JDK 1.2 and above).

Documentation

Tapestry has quite a bit of documentation available (most of these links will be broken if you did not download the full distribution).

  • Tapestry Tutorial (HTML) -- A good introduction to what Tapestry is and how to use it.
  • Primix Virtual Library Guide (PDF) -- Describes how to setup and run the Virtual Library demonstration, a complete J2EE application implemented using Tapestry.
  • Tapestry API (Javadoc)
  • Tapestry Examples (HTML) -- guide to Tapestry example and tutorial code
  • Tapestry Whitepaper (PDF) -- An executive summary of Tapestry
  • Tapestry Developer's Guide (HTML) -- main reference for designing and developing Tapestry applications
  • Java Build Environment (HTML) -- guide to setting up and using the JBE (a set of Makefiles used to build and run Tapestry and Tapestry applications).
